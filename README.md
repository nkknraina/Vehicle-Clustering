# Vehicle Clustering using Hierarchical Clustering and K-Means

## Overview
This project applies clustering algorithms to group vehicles based on their specifications.

## Dataset
- Vehicle dataset (cars.csv)

## Features Used
- engine_s
- horsepow
- wheelbas
- width
- length
- curb_wgt
- fuel_cap
- mpg

## Algorithms
1. Agglomerative Hierarchical Clustering
2. K-Means Clustering

## Evaluation Metric
- Silhouette Score

## Results
- Hierarchical Clustering: 0.36
- K-Means: 0.37

## Tools
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Google Colab
